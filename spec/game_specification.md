# 魚漢字タイピングゲーム - 企画・開発仕様書

熟練のゲームデザイナー兼プログラマーによる、子供から大人まで楽しめる「学べる」一本釣りタイピングゲームの設計仕様書です。

---

## 1. 企画概要

* **タイトル名**: 魚漢字タイピング
* **コンセプト**: 「一漁一会（いちぎょいちえ）」― 魚偏の漢字をタイピングで楽しく覚えて、勢いよく釣り上げる爽快な知育ゲーム。
* **ターゲット層**: 
  * タイピング練習をしたい子供・学生
  * 脳トレや難読漢字を楽しみたい大人・シニア層
  * 回転寿司をもっと楽しみたいすべての世代
* **主要な価値**:
  * 段階的な難易度による、挫折のない学習体験。
  * 漢字の「つくりの由来（豆知識）」を予習・復習できる親切設計。
  * 魚のシルエット（形で推測できる）と漢字を結びつける直感的なデザイン。

---

## 2. 画面構成と遷移仕様

ゲームは以下の4つのステート（画面）で構成され、直感的かつストレスなく遷移します。

### 2.1. タイトル画面（Title Screen）
* **タイトルロゴ**: 錨（アンカー）アイコンをあしらった、和風モダンかつ深海を想起させる美しいビジュアル。
* **難易度レベル選択（累積型出現システム）**:
  * **優しい（初級）**: 鮪、鮭、鯛など、食卓や回転寿司でおなじみの基本20種。
  * **普通（中級）**: 鰆、鰤、鮃など、読めそうで読めない18種に加え、**優しいレベルの魚（20種）も累積して登場（全38種）**。
  * **厳しい（上級）**: 鯒、鰰、鱏など、日常で見かけない難読20種に加え、**優しい・普通レベルの魚（38種）も累積して登場（全58種）**。
  * **エキスパート（超級）**: **全58種類すべてが登場**し、スクロールスピードが極限（通常の2.4倍）に跳ね上がる最高峰難易度。
* **最高得点（ハイスコア）表示**:
  * レベルごとに獲得したハイスコア（釣り上げた最高匹数）を個別に保存して表示。
  * プレイヤーの向上心を刺激します。ハイスコアデータは、ブラウザの `localStorage` に安全に永続化されます。
* **グローバルコントロール**:
  * ミュート設定切り替えボタン（Web Audio APIによるオリジナルシンセ効果音：タイピング間違い時や一本釣り成功時の音を鳴らすかどうかをコントロール）。
  * **漢字の振り仮名（読み・ヒント）設定トグル**: ゲーム中の漢字の上に表示される和名ヒントや、タイピングガイドに表示される「読み（ひらがな）」の表示・非表示を切り替えるスイッチ。
  * 全ハイスコアをリセットするリセットボタン。
* **アクション**: 「この難易度で予習＆スタート！」ボタンで事前予習画面へ進みます。

### 2.2. 事前予習画面（Stage Preview）
* **概要**: 本番前のインプット学習。
* **コンテンツ**: 
  * 選択した難易度で出現するすべての魚（選択レベルに応じて20〜58種類）の漢字・読み・豆知識（漢字の部首やつくりの由来）を美しくカード形式で一覧表示。
  * 事前に魚のシルエットと「ローマ字キーガイド」を見ることができ、タイピングミスを防ぐための手助けをします。
* **アクション**: 「準備完了！ゲーム本番へ」ボタンを押す（またはスペースキーを押す）と、タイピングゲームが開始されます。

### 2.3. タイピングゲーム画面（Gameplay Screen）
* **タイムリミット**: 1プレイ **50秒** の制限時間。
* **UIインジケーター**: 
  * 残り時間（TIME LEFT）：15秒以下になるとローズレッドカラーに点滅し、カウントダウンを盛り上げます。
  * 現在の釣果（スコア）：釣り上げた合計の魚の数を「〇 匹」でリアルタイム表示。
  * コンボ数（Combo）：ミスをせずに連続でタイピングを成功させると「Combo」が重なり、火の玉アイコンとともに燃え上がります。
* **魚のスクロールレーン**:
  * 画面中央を魚のシルエットが左から右へとゆっくり（またはエキスパートは高速に）泳ぎます。
  * 魚の下には「出題漢字（例：鮪）」と、その漢字の一般的な「意味・和名（例：マグロ）」がバルーンで表示されます（タイトル画面で「振り仮名」を非表示に設定している場合は「？？？」と表示されます）。
  * **無限非巡回デッキシステム（出現の完全シャッフル化）**: プレイリストが1周し終わった際、または魚を見逃した（右端まで到達）後に、全く同じ順番が巡回することを完全に防ぐ「山札（デッキ）再シャッフルシステム」を導入。デッキ内の魚種が尽きるたびに、現在の難易度のプールから完全に再シャッフルされた山札が生成されて自動補充されます。これにより、毎回違う順番で予測不可能な一本釣りを無限に楽しめます。
* **浅瀬の日差しをイメージした明るいアクアマリン背景**:
  * 暗い深海を想起させていた前バージョンから、太陽の光が差し込む明るく澄み切ったさわやかな水中（アクアブルー・エメラルドグリーン）のグラデーションに一新。
  * 明るさを引き立てるゆらめくサンライトスポットや輝く水流エフェクトを追加し、文字盤の視認性（高コントラスト）を保ちながらも、視覚的に明るく美しい海中の爽快感を表現しました。
* **ローマ字入力インターフェース**:
  * 現在入力すべきアルファベットが一文字ずつハイライトされ、すでに入力し終えた文字は「エメラルドグリーン」に変わります。
  * **ネタバレ防止型伏せ字システム（振り仮名オフ時）**: 振り仮名を「非表示」に設定している場合、キー入力プレビューに事前にローマ字の答えが表示されてしまうのを防ぐため、未入力のローマ字はすべて `?` マークの伏せ字になります（例：`koi` の場合、未入力時は `? ? ?` となり、タイピングを成功させた文字から順にアルファベットが明かされます）。これにより、漢字本来の読みを脳から引き出す真の脳トレ体験を提供します。
  * **親切設計（ノーペナルティ）**: タイピングミスが発生しても、ライフ減少や減点などのペナルティは一切ありません（コンボのみリセットされ、漢字バルーンがブルブルと小さく揺れるシャックフィードバックと、ブッという短い低音のエラー効果音が鳴ります）。バックスペースによる削除は不要で、そのまま正しいキーを押すだけで進めます。
  * **見逃し（スルー）**: 魚が右端まで到達して消えた場合も、ペナルティなしで次の魚へと自動的に切り替わります。
* **一本釣りエフェクト**:
  * タイピングをすべて正しく完了させると、金色の「釣り針と釣り糸」が上空から降りてきて、魚のシルエットを「シュッ」と勢いよく上に引き上げるアニメーションとともに、ピコーン！という心地よいCメジャーコードの爽快な上昇アルペジオ効果音が鳴り響きます。

### 2.4. リザルト・復習画面（Result Screen）
* **釣果結果表示**:
  * 「釣り上げた合計数（匹）」を中央に大きく表示。
  * プレイ中に達成した「最大コンボ数（Combo）」を表示。
  * 今回のタイピング「正確率（%）」（総打鍵数に対する正答打鍵数の割合）を算出・表示。
* **復習・リベンジおさらいメモ（別々表示システム・重複排除対応）**:
  * 今回のプレイで遭遇した魚たちが、結果に合わせて完全に別々の独立した枠に整理されて表示されます。また、同じ魚を複数回釣り上げたり逃がしたりした場合でも、リスト内には重複せず1回だけ表示される親切な重複排除設計となっています。
  * **釣り上げた魚漢字リスト（復習メモ）**: 実際にタイピングに成功して釣り上げた魚たちの漢字、読み、ローマ字、およびその魚の「豆知識（由来）」をエメラルドグリーンの美しいリストで再確認できます。ゲームを通して覚えた魚を再読することで、確実な学習定着を促します。
  * **釣り逃した魚漢字リスト（リベンジメモ）**: 画面端まで到達して逃がしてしまった魚たちの漢字、読み、ローマ字、豆知識をバラ色のリストで確認でき、「次回こそは釣り上げるぞ！」というリベンジのモチベーションを高めます。一匹も逃さなかった場合は、特別なお祝いメッセージが表示されます。
* **アクション**:
  * 「同じレベルでもう一度遊ぶ」ボタン。
  * 「レベルを変える・タイトルに戻る」ボタン。

## 3. コアメカニクス（ゲームシステム詳細）

### 3.1. 魚のシルエット表現システム（形での推測）
すべての出現魚は、その身体的な特徴に基づいて以下の6つの系統の「シルエット（ベクトルグラフィック）」に分類されます。これにより、文字だけでなく「形で魚を覚える・推測する」知育プロセスが実現します。

1. **tuna型（マグロ型）**: 鮪、鮭、鰯、鯖、鰹、鱈、鮎、鰊、鰆、鰺、鰤、鱸、鯔、鱪、鱚、鰰、鰔、鮗、鰦、すばしり、鱒、鰡、すけとうだら、さより
   * 回遊魚らしい、流線型のスリムでスポーティな体つき。
2. **round型（丸型）**: 鯛、蛸、鯉、鰻、鰍、鯰、めばる、あんこう、鮒、うきぶくろ
   * ふっくらとした丸みのある体つきや、独特の丸い頭。タコは独自のシルエット（タコ型）を表示。
3. **flat型（平ら型）**: 鰈、鮃、こち、エイ、かがみだい、ひいらぎ、まなかつお
   * 海底に這うような扁平なボディ。カレイ・ヒラメは目が片側に寄っているビジュアルを再現。
4. **shark型（サメ型）**: 鮫、チョウザメ
   * 鋭い背びれと、凶暴かつシャープな流線型の大きなボディ。
5. **shell型（貝型）**: 貝、アサリ、アワビ、かずのこ、カブトガニ、しじみ
   * 殻を持つ特徴的な二枚貝や、鎧をかぶったドーム状の形。
6. **shrimp型（エビ型）**: エビ
   * 腰が曲がり、ヒゲが長く伸びた特徴的な甲殻類のシルエット。

### 3.2. ローマ字入力・タイピング寛容システム（マルチキー入力対応）
* **マルチキー入力（打鍵揺れの完全許容）**: 
  * 日本語のローマ字入力における複数通りの打ち方を自動的に解析し、すべて許容します。
  * （例）「し」を `shi` または `si`、「ふ」を `fu` または `hu`、「ち」を `chi` または `ti`、「つ」を `tsu` または `tu`、「じ」を `ji` または `zi`、「じゃ」を `ja` または `zya` または `jya` など、すべての入力パターンに対応。
  * ユーザーがタイプした文字をリアルタイムに検知し、UI上のガイド用アルファベット配列を自動的にマッチするスタイル（例：「しじみ」において、ユーザーが `s` を打った瞬間、アルファベット盤が `shijimi` から `sijimi` へ瞬時に変更）に書き換えることで、打鍵の思考を一切邪魔しません。
* 入力キーが正しければ即座に判定が進むため、入力の遅延がありません。
* ミスをしても、プレイヤーが慌ててバックスペース（消去）を叩く必要がなく、そのまま正しいキーを押せばよいため、タイピング初学者でも快適・爽快にゲームを楽しめます。
* 一単語をコンプリートした瞬間に、即座に釣り上げモーションが発動し、次の魚がリセットされた状態で左から出現します。

## 4. 知育・学習要素（豆知識リスト）

すべての魚には、右側の漢字（つくり）の構成や性質に由来した豆知識がデータベース化されており、いつでもおさらいが可能です。
（例）
* **鮪（まぐろ）**: 「有」は丸く囲む意味を持ち、広大な海を群れを成して回遊し続ける魚を表します。
* **鮭（さけ）**: 「圭」には三角にとがった形の土器の意味があり、産卵期に鼻先が尖るサケの特徴から来ています。
* **鯛（たい）**: 「周」は平らで万遍なく行き届く意味があり、体が平たくて日本中どこでも獲れる平らな魚を表します。
* **鰯（いわし）**: 陸に上げるとすぐに弱って死んでしまうことや、他の魚の餌になりやすい「弱い魚」から「魚偏に弱」と書きます。

## 5. 技術・実装スタック

* **言語**: TypeScript (型安全かつ保守性の高いコード)
* **フロントエンド**: React 19 / Vite
* **アニメーションライブラリ**: `motion/react` (AnimatePresenceを使用した画面のスムーズなフェードイン・アウト、釣り上げ時の上昇アニメーション、間違えたときのバウンス効果などを滑らかに表現)
* **アイコンライブラリ**: `lucide-react`
* **スタイルシート**: Tailwind CSS 4 (フルレスポンシブ、深海・アクアリウムを想起させるダークネイビーとエメラルド、ネオンブルーを基調としたカラーパレット)
* **ストレージ**: Web LocalStorage (ハイスコアの永続保存)
